Shoy.Utility
============

shoy's Common Source

    version 1.0.0

常用代码类：

    Extend文件:常用的扩展，主要是string。
    
    UseTest文件:简便的测试方法。
    
    CacheCls：web缓存辅助类。
    
    CompressCls：压缩辅助类。
    
    CookieCls：Cookie操作辅助。
    
    EmailCls：发送邮件辅助。
    
    HTemplate：一个简单的html模板辅助。
    
    HttpHelper：网页爬虫辅助。
    
    ImageCls：图片处理。
    
    Utils：其他类综合辅助方法。
